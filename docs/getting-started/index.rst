Welcome to tandem!

Tandem may be used in 2D and 3D and supports low-order as well as high-order spaces.
The specific configuration is selected at compile time.
To run Tandem immediately without compiling, use the
:doc:`pre-built Docker release images <docker-release>`.
To compile Tandem from source, follow the :doc:`installation <installation>` guide.

Once tandem is installed, try to run the :doc:`example problems <examples>`.
